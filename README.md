# Self-Balancing-Robot-BLE

A two-wheeled self-balancing robot built on the Arduino Nano 33 BLE Sense for ELEC 391 (UBC, Summer 2026). Designed and documented as an embedded systems portfolio project. Features closed-loop PID balance control, encoder-based distance tracking and turning, heading hold, and full wireless control via a custom Flutter BLE app.
